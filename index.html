1. INTRODUÇÃO

Esta Política de Privacidade descreve como o aplicativo QRV Policial coleta, usa, armazena e protege suas informações pessoais. Ao utilizar nosso aplicativo, você concorda com as práticas descritas nesta política.

2. INFORMAÇÕES QUE COLETAMOS

2.1 Permissões do Dispositivo

  • Câmera (android.permission.CAMERA): Utilizada para captura de fotos, identificação facial e leitura de códigos QR para funcionalidades policiais
  • Armazenamento: Para salvar dados localmente no dispositivo
  • Internet: Para exibição de anúncios, backup na nuvem e sincronização
  • Localização (opcional): Para geolocalização de registros quando autorizado

2.2 Dados Pessoais

  • Informações inseridas pelo usuário no sistema de cadastro
  • Fotos capturadas através da câmera (armazenadas localmente)
  • Dados biométricos processados localmente para reconhecimento facial
  • Dados compartilhados voluntariamente entre usuários

2.3 Dados de Publicidade

  • Identificadores de dispositivo para personalização de anúncios
  • Informações de uso do aplicativo para otimização de anúncios
  • Dados coletados por parceiros publicitários (Google AdMob)

3. COMO USAMOS SUAS INFORMAÇÕES

3.1 Finalidades Principais

  • Funcionamento do sistema de cadastro e identificação policial
  • Processamento local de reconhecimento facial
  • Backup e sincronização de dados (Google Drive)
  • Melhoria da experiência do usuário
  • Suporte técnico e atendimento ao cliente

3.2 Publicidade

  • Exibição de anúncios personalizados
  • Sistema de créditos baseado em visualização de anúncios
  • Análise de performance publicitária

4. ARMAZENAMENTO E SEGURANÇA

Importante: O aplicativo não coleta dados pessoais dos usuários de forma automática. Os dados são inseridos voluntariamente pelo usuário.

4.1 Armazenamento Local

  • Dados armazenados localmente no dispositivo com criptografia
  • Processamento biométrico realizado inteiramente no dispositivo
  • Controle total do usuário sobre seus dados

4.2 Backup na Nuvem

  • Backup opcional e sincronizado no Google Drive da sua conta
  • Dados criptografados antes do envio
  • Acesso restrito à sua conta Google

4.3 Medidas de Segurança

  • Criptografia de dados sensíveis
  • Acesso protegido por autenticação
  • Processamento local de dados biométricos
  • Atualizações regulares de segurança

5. COMPARTILHAMENTO DE DADOS

Política de Não Compartilhamento: NÃO compartilhamos dados pessoais com terceiros, exceto nas situações descritas abaixo.

5.1 Compartilhamento Autorizado

  • Dados compartilhados voluntariamente entre usuários do aplicativo
  • Informações necessárias para funcionamento de serviços (Google Drive, AdMob)
  • Cumprimento de obrigações legais quando exigido por autoridades

5.2 Parceiros de Serviço

  • Google AdMob: Para exibição de anúncios personalizados
  • Google Drive: Para backup opcional de dados
  • Google Play Services: Para funcionalidades do sistema

6. RECONHECIMENTO FACIAL E DADOS BIOMÉTRICOS

6.1 Processamento Local

  • Todos os dados biométricos são processados localmente no dispositivo
  • Nenhum dado biométrico é enviado para servidores externos
  • Tecnologia em constante desenvolvimento e aprimoramento

6.2 Limitações e Recomendações

  • Resultados devem ser interpretados como auxílio, não como prova definitiva
  • Recomendamos sempre confirmar resultados com métodos complementares
  • A precisão pode variar dependendo das condições de iluminação e qualidade da imagem

7. SEUS DIREITOS

7.1 Controle de Dados

  • Acesso: Visualizar todos os dados armazenados no aplicativo
  • Exclusão: Deletar dados pessoais a qualquer momento
  • Portabilidade: Exportar seus dados em formato legível
  • Correção: Modificar informações incorretas

7.2 Controle de Permissões

  • Revogar permissões de câmera a qualquer momento nas configurações do dispositivo
  • Desabilitar backup na nuvem nas configurações do aplicativo
  • Controlar personalização de anúncios

8. RETENÇÃO DE DADOS

  • Dados locais: Mantidos até exclusão manual pelo usuário
  • Backup na nuvem: Mantido enquanto a conta Google estiver ativa
  • Dados de publicidade: Conforme políticas do Google AdMob
  • Logs de suporte: Mantidos por até 12 meses para resolução de problemas

9. MENORES DE IDADE

Este aplicativo é destinado a profissionais de segurança pública e não é direcionado a menores de 18 anos. Não coletamos intencionalmente informações de menores de idade.

10. TRANSFERÊNCIAS INTERNACIONAIS

Alguns dados podem ser processados em servidores localizados fora do Brasil (Google Services). Garantimos que essas transferências seguem padrões internacionais de proteção de dados.

11. ALTERAÇÕES NESTA POLÍTICA

  • Esta política pode ser atualizada periodicamente
  • Mudanças significativas serão comunicadas através do aplicativo
  • Recomendamos revisar esta política regularmente
  • Uso continuado após alterações constitui aceitação dos novos termos

12. BASE LEGAL (LGPD)

O tratamento de dados pessoais no QRV Policial baseia-se nas seguintes hipóteses legais:

  • Consentimento: Para funcionalidades opcionais como backup na nuvem
  • Execução de contrato: Para fornecimento dos serviços do aplicativo
  • Interesse legítimo: Para melhoria do aplicativo e segurança
  • Exercício regular de direitos: Para atividades de segurança pública
